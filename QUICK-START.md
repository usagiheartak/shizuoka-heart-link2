## 🚀 簡単3ステップでデプロイ

### ステップ1: ZIPファイルを解凍
このZIPファイルを解凍してください。
`shizuoka-heart-link`フォルダができます。

### ステップ2: GitHubにアップロード
1. GitHub（https://github.com）にログイン
2. 新しいリポジトリ「shizuoka-heart-link」を作成
3. フォルダの**中身**（index.htmlなど全ファイル）をアップロード
   ⚠️ フォルダごとアップロードしない！中身だけ！

### ステップ3: GitHub Pagesを有効化
1. Settings → Pages
2. Branch: **main** を選択
3. Folder: **/ (root)** を選択
4. Save をクリック

→ 1〜2分後、ウェブサイトが公開されます！

## 📝 更新方法

ファイルを変更したい場合:

1. GitHubのリポジトリページを開く
